Convert Sorted Array to Binary Search Tree

LeetCode Link: https://leetcode.com/problems/convert-sorted-array-to-binary-search-tree/

Difficulty: easy

Description

Given an integer array nums where the elements are sorted in ascending order, convert it to a height-balanced binary search tree.

A height-balanced binary tree is a binary tree in which the depth of the subtrees of every node never differs by more than one.

Test Cases

Input: nums = [-10,-3,0,5,9]
Output: [0,-3,9,-10,null,5] or [0,-10,5,null,-3,null,9]

Input: nums = [1,3]
Output: [3,1] or [1,null,3]

Input: nums = [1]
Output: [1]

Constraints

  • 1 <= nums.length <= 10^4
  • -10^4 <= nums[i] <= 10^4
  • nums is sorted in a strictly increasing order.
