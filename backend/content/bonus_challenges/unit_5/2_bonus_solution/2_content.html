Match

Common solutions to graph problems:

  1. Depth-First Search (DFS): Excellent for this problem. We can use DFS to explore each island completely and calculate its area.
  2. Breadth-First Search (BFS): Would also work by exploring islands level by level, but DFS is typically more straightforward to implement for area calculation.

Plan

Use a DFS approach

  1. Initialize a variable maxArea to track the maximum island area found
  2. Iterate through each cell in the grid
    1. If the current cell is a '1' (land) and hasn't been visited
    2. Add 1 to the current islands area
        3. Perform DFS from this cell to mark all connected land cells as visited
        1. Check if the current cell is out of bounds or water
        2. Mark the current cell as visited (change '1' to '0')
        3. Recursively call DFS on the four adjacent cells
        4. Return the calculated area
    3. Update the max area
  3. Return the max area.

Complexity Analysis

Time Complexity:

O(R x C) where R is the number of rows and C is the number of columns in the grid. We visit each cell in the grid at most once.

Space Complexity:

O(R x C) in the worst case for the recursion stack. This happens when the grid is filled with lands and DFS goes RxC levels deep.
